COMMON ERROR PATTERNS

SIGN REVERSAL ERROR 42% Frequency
Kondor frequently swaps DT/CR signs on suspense accounts (323/317) during value date processing.
Solution: Always validate DT/CR pairs. Expected: DT 323 ‚Üî CR 317 on trade date.
BUFFER ACCOUNT MISMATCH 28% Frequency
Using LCY buffer (40) for FCY transactions instead of FCY buffer (252).
Solution: FCY = Account 252, LCY = Account 40. Never mix currencies.
DELTA CODE CONFUSION 18% Frequency
Wrong delta codes: 934 (Trade) vs 936 (Value) vs 938 (Accrual) vs 940 (Maturity).
Solution: Memorize delta code sequence. Each lifecycle stage has unique code.
